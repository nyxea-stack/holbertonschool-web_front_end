# Flexbox Project

## Description
This project focuses on learning and practicing CSS Flexbox. I use Flexbox to build layouts, replace float-based positioning, and align elements properly in a responsive way using only HTML and CSS.

## Learning Objectives
By the end of this project, I am able to:
- Explain what Flexbox is and why it is useful
- Convert layouts based on floats to Flexbox
- Align elements horizontally and vertically using Flexbox
- Understand the difference between the main axis and the cross axis
- Distinguish properties applied to flex containers from those applied to flex items
- Use flex shorthand properties
- Build a page with Flexbox in mind from the start
